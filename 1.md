
# DevOps Lab: Git Commands Demonstration

## Aim

To demonstrate and create a project in **local and remote repository** using Git Bash and GitHub and apply Git commands:
`init`, `status`, `log`, `add`, `commit`, `push`, `config`, `clone`, and `reset`.

---

# Requirements

* Git Bash Installed
* GitHub Account
* Internet Connection
* Text Editor (Notepad / VS Code)

---

# Procedure

## Step 1: Configure Git User Information

Set username and email (only first time setup).

```bash
git config --global user.name "chiranthgangadakar-j"
git config --global user.email "vvce23cseaiml0077@vvce.ac.in"
```

### Verify Configuration

```bash
git config --list
```

### Sample Output

```text
user.name=JayanthR5
user.email=jayanthr5@gmail.com
core.editor=vim
```

---

# Step 2: Create Local Repository

Create a project folder.

```bash
mkdir git-lab
cd git-lab
```

Initialize Git repository.

```bash
git init
```

### Sample Output

```text
Initialized empty Git repository in /git-lab/.git/
```

---

# Step 3: Check Repository Status

```bash
git status
```

### Sample Output

```text
On branch main

No commits yet

nothing to commit (create/copy files and use "git add" to track)
```

---

# Step 4: Create a File

Create a text file.

```bash
echo "This is my first Git Lab file" > demo.txt
```

Check status again.

```bash
git status
```

### Sample Output

```text
Untracked files:
  demo.txt
```

---

# Step 5: Add File to Staging Area

```bash
git add demo.txt
```

Or add all files:

```bash
git add .
```

Check status:

```bash
git status
```

### Sample Output

```text
Changes to be committed:
  new file: demo.txt
```

---

# Step 6: Commit the File

```bash
git commit -m "First commit - Added demo.txt"
```

### Sample Output

```text
[main (root-commit) abc1234] First commit - Added demo.txt
 1 file changed, 1 insertion(+)
 create mode 100644 demo.txt
```

---

# Step 7: View Commit History (git log)

```bash
git log
```

### Sample Output

```text
commit abc1234
Author: Jayanth R
Date: Sun Mar 22

First commit - Added demo.txt
```

---

# Step 8: Create Remote Repository in GitHub

1. Open GitHub
2. Click **New Repository**
3. Repository Name:

```text
git-lab
```

4. Click **Create Repository**

Copy repository URL:

```text
https://github.com/JayanthR5/git-lab.git
```

---

# Step 9: Add Remote Repository

```bash
git remote add origin https://github.com/JayanthR5/git-lab.git
```

Verify remote:

```bash
git remote -v
```

### Sample Output

```text
origin https://github.com/JayanthR5/git-lab.git (fetch)
origin https://github.com/JayanthR5/git-lab.git (push)
```

---

# Step 10: Push Code to GitHub

```bash
git push -u origin main
```

### Sample Output

```text
Enumerating objects: 3, done.
Counting objects: 100% (3/3), done.
Writing objects: 100% (3/3), done.
Branch 'main' set up to track 'origin/main'.
```

---

# Step 11: Clone Remote Repository

Move outside folder:

```bash
cd ..
```

Clone repository:

```bash
git clone https://github.com/JayanthR5/git-lab.git
```

### Sample Output

```text
Cloning into 'git-lab'...
remote: Enumerating objects
Receiving objects: 100%
```

---

# Step 12: Modify File and Commit Again

Open file and add text:

```bash
echo "Second line added" >> demo.txt
```

Add file:

```bash
git add demo.txt
```

Commit changes:

```bash
git commit -m "Updated demo.txt file"
```

---

# Step 13: Demonstrate Git Reset

## Soft Reset

Undo last commit but keep changes.

```bash
git reset --soft HEAD~1
```

### Sample Output

```text
Changes to be committed:
 modified: demo.txt
```

---

## Hard Reset

Remove last commit and changes permanently.

```bash
git reset --hard HEAD~1
```

### Sample Output

```text
HEAD is now at abc1234 First commit
```

---

# Step 14: Push Changes Again

```bash
git push origin main
```

---

# Git Commands Used

| Command        | Description            |
| -------------- | ---------------------- |
| git config     | Set username and email |
| git init       | Initialize repository  |
| git status     | Check file status      |
| git add        | Add files to staging   |
| git commit     | Save changes           |
| git log        | Show commit history    |
| git remote add | Connect to GitHub      |
| git push       | Upload to GitHub       |
| git clone      | Copy repository        |
| git reset      | Undo changes           |
