
# DevOps Lab: Fork, Branch, Merge, Diff, Merge Conflict and Pull Request Using GitHub and Git Bash

---

# Aim

To demonstrate creation of a project in **remote repository** and apply **fork, branch, diff, merge, merge conflict, and pull request** concepts using **GitHub and Git Bash**.

---

# Requirements

* Two GitHub Accounts

  * Original Repository Account
  * Fork Account → **JayanthR5**
* Git Bash Installed
* Internet Connection
* Web Browser

---

# Assumption

Original repository exists:

```text
https://github.com/original-user/git-lab
```

Fork repository:

```text
https://github.com/JayanthR5/git-lab
```

---

# Procedure

# Step 1: Fork Repository from Original Account

Login to GitHub using:

```text
Username: JayanthR5
```

Open original repository:

```text
https://github.com/original-user/git-lab
```

Click:

```text
Fork
```

### Sample Output

```text
Fork created successfully
Repository copied to JayanthR5 account
```

Forked repository:

```text
https://github.com/JayanthR5/git-lab
```

---

# Step 2: Clone Fork Repository Using Git Bash

Open Git Bash.

```bash
git clone https://github.com/JayanthR5/git-lab.git
```

### Sample Output

```text
Cloning into 'git-lab'...
remote: Enumerating objects
Receiving objects: 100%
```

Move into directory:

```bash
cd git-lab
```

---

# Step 3: Create Branch Using Git Bash

```bash
git branch feature1
```

Switch branch:

```bash
git checkout feature1
```

Or:

```bash
git checkout -b feature1
```

### Verify Branch

```bash
git branch
```

### Sample Output

```text
* feature1
  main
```

---

# Step 4: Modify File and Show Diff

Modify file:

```bash
echo "Feature1 changes added from branch" >> demo.txt
```

Check difference:

```bash
git diff
```

### Sample Output

```text
+ Feature1 changes added from branch
```

---

# Step 5: Add and Commit Changes

```bash
git add demo.txt
```

```bash
git commit -m "Feature1 branch update"
```

### Sample Output

```text
[feature1 abc123]
1 file changed, 1 insertion(+)
```

---

# Step 6: Push Branch to GitHub

```bash
git push origin feature1
```

### Sample Output

```text
Branch 'feature1' pushed successfully
```

---

# Step 7: Create Pull Request Using GitHub

Go to GitHub.

Open:

```text
https://github.com/JayanthR5/git-lab
```

Click:

```text
Compare & Pull Request
```

Set:

```text
Base Repository: original-user/git-lab
Head Repository: JayanthR5/git-lab
```

Click:

```text
Create Pull Request
```

### Sample Output

```text
Pull request created successfully
```

---

# Step 8: Merge Pull Request

Login to:

```text
Original Account
```

Click:

```text
Merge Pull Request
```

Then:

```text
Confirm Merge
```

### Sample Output

```text

```

---

# Step 9: Demonstrate Merge Conflict Using Git Bash

Switch to main branch:

```bash
git checkout main
```

Create another branch:

```bash
git checkout -b feature2
```

Modify same file:

```bash
echo "Feature2 conflicting line" >> demo.txt
```

Add and commit:

```bash
git add demo.txt
git commit -m "Feature2 change"
```

Switch back to main:

```bash
git checkout main
```

Modify same file again:

```bash
echo "Main conflicting line" >> demo.txt
```

Commit:

```bash
git add demo.txt
git commit -m "Main update"
```

Merge branch:

```bash
git merge feature2
```

### Sample Output

```text
CONFLICT (content): Merge conflict in demo.txt
Automatic merge failed
```

---

# Step 10: Resolve Merge Conflict

Open `demo.txt`.

You will see:

```text
<<<<<<< HEAD
Main conflicting line
=======
Feature2 conflicting line
>>>>>>> feature2
```

Edit file:

```text
Main conflicting line
Feature2 conflicting line
```

Save file.

Add file:

```bash
git add demo.txt
```

Commit:

```bash
git commit -m "Merge conflict resolved"
```

### Sample Output

```text
Merge conflict resolved successfully
```

---

# Step 11: Demonstrate Git Diff Between Branches

```bash
git diff main feature1
```

### Sample Output

```text
Difference between branches displayed
```

# Concepts Used

| Concept        | Description                          |
| -------------- | ------------------------------------ |
| Fork           | Copy repository from another account |
| Branch         | Create separate working branch       |
| Diff           | Show changes between files           |
| Merge          | Combine changes                      |
| Merge Conflict | Conflict when same lines modified    |
| Pull Request   | Request to merge branch              |
